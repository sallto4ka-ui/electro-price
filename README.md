Актуальный прайс: Разделен по категориям (Черновые, Чистовые, Щит).
Интерактивность: Можно менять количество (метров, штук) прямо в списке.
Корзина (Смета): Закреплена справа (на ПК) или снизу (на мобильном), обновляется мгновенно.
Функция заказа: Кнопка "Заказать в WhatsApp" формирует готовое сообщение со списком работ.
